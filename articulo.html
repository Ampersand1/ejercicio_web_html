WebHOLE: Developing a web-based hands-on learning environment to assist beginners in learning web application security

Ficha Técnica del Artículo

Resumen

With the rapid growth of web applications, web application security (WAS) has become an important cybersecurity issue. For efective WAS protection, it is necessary to cultivate and train personnel, especially beginners, to develop correct concepts and practical hands-on abilities through cybersecurity education. At present, many methods ofer vulnerable web environments to support practical hands-on training, including large-scale “Capture the Flag” mode (e.g., Cyber Range), preconfgured virtual machine images (e.g., Mutillidae), pre-built stand-alone applications (e.g., WebGoat), and web-based system (e.g., Damn Vulnerable Web Application). However, beginners need not only hands-on training tools and systems but also assistance to support efective learning. Moreover, pre-built training content and exercises are usually not easy to modify and thus lack the fexibility to meet specifc teaching needs. Therefore, this study proposed and developed the Webbased Hands-On Learning Environment (WebHOLE) to efciently assist beginners in learning WAS. To improve the fexibility of the training content, a web-based authoring tool was developed in WebHOLE to create customized hands-on learning exercises. Accordingly, learners can learn and practice the WAS training content online with learning assistance provided by the hands-on learning system. The hands-on abilities of the learners can be efciently assessed by the hands-on testing system using online exams with progressive hints and automatic grading. Furthermore, to improve the efectiveness of teaching and testing, a portfolio analysis scheme using a data mining technique was developed to identify learning barriers and problematic test items. WebHOLE was applied to an actual beginner-level WAS course for undergraduate students. The experimental results showed the benefts of WebHOLE on WAS learning, with a signifcant improvement in learning outcomes. Students expressed high satisfaction with WebHOLE’s learning assistance, rating it with average satisfaction scores above 4.0 out of 5.0. The portfolio analysis scheme also showed the efectiveness of WebHOLE in identifying learning problems and refning test items.

Ideas Principales

1. Necesidad de formación en seguridad de aplicaciones web (WAS): Con el crecimiento de las aplicaciones web y el aumento de los ataques cibernéticos como inyecciones SQL y ataques XSS, es crucial entrenar a principiantes en seguridad cibernética, desarrollando sus habilidades prácticas a través de ambientes de aprendizaje prácticos y personalizables.

2. Desarrollo del entorno WebHOLE: El artículo presenta WebHOLE, un entorno de aprendizaje práctico en línea diseñado para ayudar a los principiantes a aprender sobre la seguridad de aplicaciones web. WebHOLE incluye herramientas para crear ejercicios personalizados, un sistema de aprendizaje asistido, y un sistema de pruebas con retroalimentación automática para mejorar la efectividad del aprendizaje.

3. Evaluación positiva de WebHOLE: Se implementó WebHOLE en un curso universitario, y los resultados experimentales mostraron que mejoró significativamente el aprendizaje y la satisfacción de los estudiantes, destacando la utilidad del análisis de portafolio para identificar barreras de aprendizaje y mejorar los elementos de prueba.

Conclusiones

En conclusión, se selecciona este artículo porque propone una experiencia práctica de ciberseguridad basada en la web para principiantes. Teniendo en cuenta la importancia de la seguridad web y la falta de aprendizaje en estos conceptos en la mayoría de los principiantes, este artículo enriquece los conocimientos en el desarrollo web debido a las múltiples referencias que hace a otras prácticas open-source de entrenamiento en ciberseguridad que se han realizado anteriormente, como Mutillidae o Metasploitable, las cuales pueden ser utilizadas para el aprendizaje en el área, al mismo tiempo que explica la metodología empleada en su propia versión, introduciendo al lector a diferentes conceptos útiles.
